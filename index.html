Flock by Block

Alina Scantlebury · Data Visualization · Spring 2026

How does San Francisco's urban infrastructure shape which birds live where? This project uses over 600,000 eBird observations to follow four species across the city and ask why each one is where it is.

Four chapters

  • Crows — .
  • Parrots — exploring parrots sightings traveling from Telegraph Hill to Fort Mason.
  • Hawks — investigating eagle raptor sightings considering the Pacific Flyway which runs through the city every autumn.
  • Shorebirds — inspecting decrease in population.

Data

  • eBird Basic Dataset · Cornell Lab of Ornithology · 621,286 records
  • SF Street Tree List · DataSF
  • SF Registered Businesses · DataSF

Built with

  • D3.js · Leaflet · Scrollama · Python · geopandas
